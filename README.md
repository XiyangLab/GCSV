# GCSV
The repository stores scripts for the project of "GCSV" - A global atlas of deep-sea cold seep viruses uncovers extensive genomic novelty and biotechnological potential.

Scripts are placed in the following folders:
- `01_viral_identification/`: viral detection, CheckV refinement, ANI clustering
- `02_provirus_cleanup/`: original provirus-boundary / IRanges / rRNA-cleanup script plus template
- `03_taxonomy/`: taxonomy assignment and consensus
- `04_host_prediction/`: iPHoP and host integration
- `05_diversity/`: Bowtie2 / MetaPop / CoverM plus original diversity plotting scripts
- `06_vpc_annotation/`: gene prediction, vPC clustering, Foldseek / eggNOG annotation
- `07_hallmark_proteins/`: MCP / TerL property calculation
- `08_amg/`: DRAM-v, VIBRANT AMG curation, hydrocarbon-gene screening
- `09_antidefense/`: DefenseFinder / dbAPIS and anti-defense curation
- `10_lysins/`: reconstructed lysin workflow templates
